使用SSR与V2Ray科学上网

· 2019-05-21 · # VPS # 科学上网

墙角数枝梅，凌寒独自开。

机场

所谓机场，就是指一个服务商购买大量的VPS搭建SSR/V2Ray，并按月/流量计费。因为VPS被称为“小鸡”，众多“小鸡”就成为了机场。一个用户连接SSR时只会占用一台VPS的一个端口，所以机场主会为每个用户分配一个端口号，这样多用户就互不影响。 与个人搭建相比，机场的优势在于，机场主会租用世界各地的VPS，主要有美国，韩国，日本，香港，新加坡等，所以用户会有多个选择。如果选择个人购买VPS搭建的话，就只有单个选择。除此之外，机场主还会为自己的节点套上CND，这样就可以极大提高网速，使用香港，新加坡这些离我们比较近的国家节点时，延迟往往在几十毫秒。

购买机场

因为机场主们都担心自己会被查水表，所以机场的网站域名经常深藏不露，而且会经常更换，机场主们选择在Telegram里面建立自己的频道，方便更换域名时通知用户。大型的机场注册会需要邀请码，因为他们已经有稳定的客户，不希望那些白嫖的用户占用账号。 所以购买机场服务的最佳去处就是Telegram，可以选择添加一些导航机器人，这些机器人会向用户推广频道。机场主有时也会使用机器人打广告。 价格一般5-15元一个月，包年的会有优惠。因为有一定风险，机场主也有跑路的可能，所以最好的选择是按月按季度购买。

公益机场

有一些古道热肠的大神们天天在为科学上网事业做出贡献，因此他们会开一些公益机场，但是IP也经常被屏蔽。 这种公益机场，一般在Telegram上也能找到。 我之前一直关注一个叫做“翻个墙”的网站（已被墙），现在只能通过修改host文件来访问，具体IP如下： 里面会分享一些公益节点及SSR的搭建方法。

自己搭建

准备工作

首先引用SSR中文网（https://ssr.tools/） 的一段话（V2Ray同理）：

SSR本质上是一种加密代理方式，其服务器搭建在外国的VPS服务器上，客户端在用户终端设备上。服务器作为中间桥梁，使我们间接访问国外网站。同时由于是加密代理，防火墙无法识别内容，访问比较安全。

首先就是要购买一台国外的VPS。VPS要求不高一般512M内存的就够我们使用了。一般年付20美刀，遇到节日还会有特价。具体的知名VPS有搬瓦工（这个网站还为我国用户打造了自带SSR的服务器，看来是调查了市场），Vultr，Linode等，支付方式有支付宝，PayPal，信用卡（Visa，万事达）等。 对了，谷歌云和AWS如果绑定信用卡可以免费试用一年哦。 我初次入坑是在美国黑色星期五促销时，在VirMach购买了一台3G内存的VPS，年付20美刀，不过服务器在纽约，有300ms的延迟。

SSR脚本

前人栽树后人乘凉，现在已经有了方便的Shell脚本供我们使用。 购买VPS后，推荐使用Debian操作系统，因为很多脚本都是要求Debian，方便开启BBR加速。

BBR是由谷歌提供的新的TCP拥塞控制算法，其目的就是要尽量跑满带宽，并且尽量不要有排队的情况

以翻个墙网站脚本为例：

#安装脚本
wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

因为脚本界面全是中文，所以可以傻瓜式安装。

  1. 输入 1 点击回车键开始安装 SSR，再点击回车键自动检测服务器 IP，然后根据需要填写 SSR 用户名和密码，端口 6 位数以内。
  2. 加密方式选择：推荐选择比较复杂的方式，降低被墙概率，如15/2/1。
  3. SSR 账号速度限制、流量限制、同时在线设备数量限制以及端口限制，根据自己的需要添加或全部点击回车键默认不做限制。
  4. 安装完成之后可以开启BBR加速。
bash ssr.sh

使用这条命令可以打开主菜单。选择14其他功能，再选择1安装BBR，最后重启服务器即可。

脚本安装完成后可以选择生成二维码。 安卓手机可以搜索并下载SSR客户端（小飞机），IOS可以下载小火箭（ShadowRocket）。扫描二维码即可连接服务器。

二者比较

个人感觉，SSR客户端比V2Ray更加友好，而且支持订阅。 但是现在SSR已经频繁被墙，而V2Ray可以选择多种协议的特点可以有效防止被墙。

  • VPS的IP被墙
  • 服务器端口被墙
  • TCP阻断
