使用Ubuntu安装IntelliJ IDEA并创建桌面图标

· 2017-11-15 · # Linux

本文主要记录我在Ubuntu操作系统下安装必要的开发软件IntelliJ IDEA以及Postman，并创建桌面图标的过程。

1.官网下载IDEA为tar.gz压缩包，IDEA解压后需要运行bin目录下的idea.sh脚本启动安装程序，并且以后运行时也需在终端启动，非常麻烦。但是可以选择创建桌面图标。

2.创建桌面图标。Ubuntu的软件快捷方式均在usr/share/applications目录下，以idea为例，在该目录下运行vim

sudo vim /usr/share/applications/intellij-idea.desktop

创建新的文件并在文件中复制以下内容：

[Desktop Entry]
Name=IntelliJ IDEA
Exec=/你的目录/idea-IU-141.1532.4/bin/idea.sh
Comment=IntelliJ IDEA
Icon=/你的目录/idea-IU-141.1532.4/bin/idea.png
Type=Application
Terminal=false
Encoding=UTF-8

其中Exec=为使用终端打开idea的方式，icon是图标所在目录。

并给该文件赋予权限:

sudo chmod +x /usr/share/applications/intellij-idea.desktop

即可得到idea的快捷方式。

将快捷方式图标拖入左侧快捷栏或者桌面即可。
