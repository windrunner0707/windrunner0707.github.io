Git学习之路

· 2019-05-19 · # Git

Github上的git学习动画： https://learngitbranching.js.org/?demo

日常工作中的一些操作：

git commit --amend 可以对上一次的提交做修改

git合并多次提交：

1.git log 查看提交历史； 2.git rebase -i HEAD~3 选取最近的几次提交，-i 指从最新提交开始计数； 3.在vi编辑模式中把需要合并提交的 pick 改为 squash，指合并到上次提交，保存； 4.进入commit merge界面，编辑合并提交信息，保存； 5.git log发现多次提交已合并； git pull --rebase 保证提交为直线，避免merge

回退上一个提交

commit回退：git reset --hard HEAD^ git reset –hard 修改远程： git push origin master -f //查看仓库所使用的连接形式 git remote -v //修改仓库的连接方式为ssh git remote set-url origin git开头的仓库地址

Git给提交加tag并提交到远程仓库

git tag —list git ls-remote —tags git tag v1.0.0 -m ‘123’ id git push tags

手动修改gitconfig文件的内容

vi .git/config
